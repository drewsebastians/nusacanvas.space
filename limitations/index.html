Terakhir ditinjau: 2026-07-12

Batasan data dan aplikasi

Mapnesia dibuat untuk peta presentasi dan laporan ringan. Peta ini bukan dokumen hukum, bukan sistem GIS resmi, dan bukan klaim batas administrasi terbaru.

Snapshot 2020

Geometri memakai snapshot ADM2 2020 dari geoBoundaries/HDX. Struktur provinsi Indonesia saat ini dicatat terpisah dalam registry v1.

519 fitur geometri

Angka 519 adalah jumlah fitur pada file geometri, bukan klaim jumlah kabupaten/kota otonom terkini.

53 metadata ambigu

Nama wilayah yang tidak bisa dipasangkan secara aman tidak dipaksa. Warnanya tetap bisa digunakan, tetapi metadata resminya perlu review.

Ekspor visual

SVG/PNG memakai proyeksi browser untuk visualisasi, bukan pengukuran presisi atau penetapan batas. PDF saat ini adalah raster, bukan PDF vektor dengan teks selectable.

Batasan performa

Startup normal memakai geometri sederhana agar ringan. Ekspor detail memuat file lokal sekitar 10,5 MB hanya setelah dipilih pengguna dan dapat menggunakan memori lebih besar.

Batasan spreadsheet

Mapnesia tidak menggabungkan beberapa baris sumber ke satu wilayah secara otomatis. Baris duplikat harus diselesaikan atau diabaikan pengguna. Guard XLSX membantu membatasi parsing browser, tetapi bukan pemindai malware. Project besar dapat melebihi localStorage; penyimpanan IndexedDB belum tersedia.

Batasan hukum dan lisensi

Atribusi sumber data harus dipertahankan. Teks privacy/terms pada staging adalah draft operasional dan tidak menggantikan nasihat hukum profesional.
