Worked example: peta nilai dua kota

Contoh ini menggunakan data sintetis Surabaya dan Denpasar agar alur dapat diuji tanpa membagikan data bisnis atau pribadi.

1. Unduh atau copy data

Unduh contoh CSV atau contoh TSV. Kolomnya adalah wilayah, provinsi, nilai, kategori, sumber, dan periode.

2. Pratinjau dan terapkan

Di tahap Input, pilih file atau paste tabel. Sistem menyarankan kolom wilayah, provinsi, nilai, kategori, sumber, dan periode. Pratinjau harus menunjukkan dua baris valid sebelum tombol Terapkan Hasil Valid dipilih.

3. Buat warna

Pilih Interval sama dengan dua atau tiga kelas, lalu tekan pratinjau dan terapkan. Tabel, peta, dan legenda akan memakai assignment yang sama. Jika Anda menghapus nilai, baris itu seharusnya masuk ke Tidak ada data—bukan warna nol.

4. Ekspor

Isi sumber “Contoh sintetis Mapnesia” dan periode “2025”, pilih Seluruh Indonesia, lalu ekspor SVG atau PNG. Anda juga dapat mengunduh PDF raster dan mapping CSV untuk memeriksa status kecocokan.

Hasil yang diharapkan

Dua wilayah cocok tanpa resolve manual, legenda mencantumkan kelas nilai, dan atribusi boundary tetap tampil pada ekspor. Tidak ada data contoh yang dikirim dari browser.
