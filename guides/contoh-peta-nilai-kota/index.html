Example: Map values for two cities

This example uses synthetic data for Kota Surabaya and Kota Denpasar. You can test the full workflow without sharing business or personal data.

1. Download or copy the data

Download the CSV sample or TSV sample. The columns are region, province, value, category, source, and period.

2. Review and add the data

Paste the table or upload the file. Confirm the suggested columns. The preview should show two valid rows before you choose Use valid rows.

3. Color by value

Choose Equal ranges with two or three groups. Review the colors, then create the map. The table, map, and legend will use the same settings. If you remove a value, that row should appear as No data, not as zero.

4. Export map

Enter “NusaCanvas synthetic example” as the source and “2025” as the period. Choose all of Indonesia, then export SVG or PNG. You can also download a raster PDF or match-report CSV.

Expected result

Both cities should match without a manual correction. The legend should show the value groups, and the required boundary attribution should remain in the export. No sample data is sent from your browser.
