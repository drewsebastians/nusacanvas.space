Make a clear map legend

A legend should explain the colors, method, and data limits. It should do more than decorate the map.

Use a specific title

Name the measure shown by the colors. Add the source and period to the export details instead of using them as category names.

Keep empty values separate from zero

NusaCanvas adds a No data entry when a row is empty, is not valid, has no safe match, or is ignored. Do not describe these rows as zero.

Choose the right color method

Equal ranges show the size of values. Ranked groups show relative position. Categories show labels. Review the legend before creating the map, and avoid using more color groups than readers can understand.

You can reverse the order of a local color palette without changing the groups. Color is not the only way to explain meaning, so keep every legend label clear.
