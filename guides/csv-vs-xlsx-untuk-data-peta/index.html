CSV vs XLSX untuk data peta

Keduanya dapat dipakai; pilih format yang paling mudah Anda periksa sebelum impor.

CSV/TSV

Ringan, mudah dibagikan, dan biasanya paling cepat untuk tabel satu sheet. CSV memakai koma; TSV memakai tab sehingga nyaman untuk paste dari spreadsheet.

XLSX

Cocok bila Anda memiliki beberapa sheet atau perlu mempertahankan workbook. Mapnesia meminta Anda memilih sheet dan memeriksa struktur ZIP sebelum parser dijalankan.

Praktik aman

  • Gunakan satu baris header dan satu tabel per sheet.
  • Jangan gunakan file Excel untuk menjalankan makro; workbook bermakro, objek tertanam, external link, dan format tak didukung akan ditolak.
  • Pastikan angka memakai format yang konsisten. Di opsi lanjutan, pilih Indonesia atau Internasional bila deteksi otomatis ambigu.

Batas pemrosesan

Parser browser memiliki batas ukuran, jumlah baris, kolom, sel, dan rasio ZIP untuk menjaga perangkat tetap responsif. Ini adalah safeguard pemrosesan, bukan klaim perlindungan malware.
