CSV or XLSX for map data?

Both formats work. Choose the one that is easiest for you to check before adding the data.

CSV or TSV

These formats are lightweight, easy to share, and usually fastest for one table. CSV uses commas. TSV uses tabs, which makes it convenient when you paste data from a spreadsheet.

XLSX

Use XLSX when your workbook has several sheets or you need to keep the workbook format. NusaCanvas asks you to choose a sheet and checks the file structure before reading the table.

Safe file practices

  • Use one header row and one table on each sheet.
  • Do not use the app to run Excel macros. Files with macros, embedded items, external links, or unsupported formats are rejected.
  • Use one number format throughout the file. In Advanced options, choose Indonesian or international number formatting if automatic detection is uncertain.

Processing limits

The app limits file size, rows, columns, cells, and compressed-file ratios to keep your device responsive. These checks do not scan a file for malware.
