FAQ

Frequently asked questions

Is my spreadsheet uploaded anywhere?

No. Spreadsheet matching, map design, and project backup happen in this browser on your device.

Why did a region name not match?

Names may use a different spelling, administrative type, or code. Review the suggested matches or follow the region-name guide.

Is my work saved?

NusaCanvas keeps a working copy in this browser and restores it automatically. Download a project file if you need a durable copy or want to move to another device.

Can I switch between spreadsheet mapping and Highlight regions?

Yes. Use the switch link in either workflow or choose Change goal in the workspace header. Existing highlights stay in the current project.

Which export format should I use?

Use SVG for editable presentation graphics, PNG for quick sharing, and PDF for documents or printing. See the export guide.

Why can the number of mapped regions differ from a current legal count?

The map geometry and administrative-name list have different reference dates. Read the region-count explanation for the exact scope.
