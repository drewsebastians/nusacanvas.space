Equal ranges or ranked groups?

Both methods divide values into color groups, but they show different patterns.

Equal ranges

The distance from the lowest value to the highest value is divided into ranges of the same width. Use this method when readers need to compare values on one consistent scale. If one value is much higher than the rest, many regions may fall into the lowest group.

Ranked groups

Ranked groups, also called quantiles, divide the ordered values so each group contains a similar number of regions. Use this method to compare relative position rather than the size of each difference. Equal values stay in the same color group, so the final number of groups may be lower than requested. NusaCanvas shows a warning when this happens.

Choose a method you can explain

  • Choose equal ranges to show the value scale.
  • Choose ranked groups to highlight relative rank.
  • Use manual ranges when a policy or business threshold is already defined.
  • Color around a midpoint when zero or a target has a special meaning.

Empty values, values that are not valid, uncertain matches, and unmatched regions are not treated as zero. They appear as No data.
