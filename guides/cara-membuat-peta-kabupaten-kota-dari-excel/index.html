Create a regency and city map from Excel

Add a region-name column and a value column, then check every match before adding colors to the map.

Prepare a simple table

Use columns for region, province, and value. A province column is useful when two regencies or cities share a name. Try the synthetic CSV sample or synthetic TSV sample.

Four steps

  1. Add data: paste a table from Excel or Google Sheets, or upload a local CSV, TSV, or XLSX file.
  2. Review columns: check the preview and confirm which columns contain regions, provinces, and values.
  3. Match regions: fix names with more than one match, or add a province or region code.
  4. Design map: choose a color method, review the result, and then export the map.

Privacy and limitations

Your file is processed in your browser and is not uploaded. The map uses a 2020 boundary snapshot for visual reference. It is not a legal boundary record or a current official administrative list.
