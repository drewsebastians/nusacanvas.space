Why map region counts can differ

Region counts often differ because the sources count different things. A map file can count boundary shapes, while an official list counts administrative units from a particular year.

What does 519 mean in NusaCanvas?

The geoBoundaries/HDX 2020 source contains 519 second-level administrative boundary shapes (ADM2) for Indonesia. One mapped shape can be a single polygon or a group of polygons. It is not always the same as one current autonomous regency or city.

Why is there no single number?

Indonesia's administrative structure changes over time. Map sources may also include administrative cities, administrative regencies, island areas, or a snapshot made before a province was divided. A 2020 map, a current official list, and regional-autonomy statistics can therefore show different totals without any one of them being automatically wrong.

Examples that are not one-to-one

  • The 2020 snapshot does not always match the current structure of 38 provinces.
  • DKI Jakarta contains administrative units that are useful on a map but do not have the same status as ordinary autonomous regencies and cities.
  • Some region names appear more than once in the source information. The app leaves 53 records without forced official details until stronger evidence is available.
  • Administrative changes in the Papua region mean that historical boundaries and current information must remain separate.

How to use this map

Use NusaCanvas as a visual tool. For legal decisions, regulations, or the latest official data, check the relevant official source. NusaCanvas includes the boundary version, region-list version, and attribution so the map keeps its context when exported.

Batch 1 principle

The project does not change totals or remove small islands to match a popular number. Differences are recorded, tested, and explained.
