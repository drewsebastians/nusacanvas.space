Cara ekspor peta ke PowerPoint

Pilih format dan metadata sebelum menyimpan peta untuk slide atau laporan.

Pilih format

Gunakan SVG bila Anda ingin mempertahankan vektor untuk PowerPoint atau editor grafis. Gunakan PNG untuk gambar siap pakai. PDF A4/A3 tersedia sebagai keluaran raster lokal untuk cetak atau berbagi; teksnya belum selectable.

Pilih cakupan

Seluruh Indonesia memakai cakupan nasional. Tampilan peta saat ini mengikuti bounds peta yang sedang Anda lihat. Pilihan ini eksplisit agar tidak terjadi ekspor area kecil tanpa sengaja.

Lengkapi metadata

Isi judul, sumber angka, dan periode. Atribusi boundary wajib tetap ditambahkan oleh aplikasi. Periksa judul legenda, catatan kaki, dan ukuran sebelum ekspor.

Ekspor peta adalah referensi visual, bukan dokumen penetapan batas hukum. Untuk gambar besar, browser dapat meminta konfirmasi atau menggunakan fallback resolusi.

Buka ekspor

Draft pratinjau staging; tidak diindeks.
