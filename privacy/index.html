Kebijakan privasi

Halaman ini menjelaskan perilaku privasi aplikasi staging. Ini bukan nasihat hukum profesional.

Data pengguna

File CSV, file proyek JSON, pilihan warna, kategori, dan nilai diproses di browser. Aplikasi tidak memiliki backend, akun, database, analytics, iklan, atau penyimpanan cloud proyek.

Penyimpanan lokal

Autosave memakai penyimpanan lokal browser. Data tersebut tetap di perangkat/browser pengguna dan dapat hilang jika data situs dibersihkan.

Ekspor dan unduhan

SVG, PNG, laporan migrasi, dan template laporan data dibuat di browser. Pengguna mengontrol sendiri file yang diunduh atau dibagikan.

Permintaan jaringan

Startup normal memuat file statis dari domain staging yang sama: HTML, CSS, JavaScript, Leaflet lokal, dan GeoJSON produksi sederhana. Geometri detail lokal hanya dimuat jika pengguna memilih ekspor detail.

Log hosting

Cloudflare sebagai penyedia hosting dapat memproses log teknis biasa untuk penyediaan layanan. Aplikasi ini sendiri tidak menambahkan pelacakan pihak ketiga.

Kontak

Belum ada kanal kontak publik yang terverifikasi di repository. Gunakan template laporan data untuk menyiapkan laporan yang dapat dikirim manual setelah owner menetapkan tujuan kontak publik.
