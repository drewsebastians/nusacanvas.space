Terakhir ditinjau: 2026-07-12

Ketentuan penggunaan

Ketentuan ini adalah teks operasional staging, bukan nasihat hukum profesional.

Lingkup penggunaan

Mapnesia disediakan untuk membuat peta warna wilayah sebagai referensi visual. Hasil ekspor tidak boleh dianggap sebagai penetapan batas hukum, data survei, atau sumber resmi tunggal.

Tanggung jawab pengguna

Pengguna bertanggung jawab memeriksa kembali data, atribusi, konteks penggunaan, dan kelayakan hasil ekspor sebelum dipublikasikan atau dipakai dalam keputusan penting.

Data dan lisensi pihak ketiga

Kode aplikasi berlisensi MIT, tetapi data batas dan library pihak ketiga tetap mengikuti lisensi sumbernya. Ringkasan lisensi ada di Sumber & Lisensi dan `ATTRIBUTION.md`.

Batasan layanan

Aplikasi staging ini tidak menyediakan akun, backup cloud, dukungan resmi, analytics, iklan, atau jaminan ketersediaan. Cloudflare workers.dev tetap noindex selama staging.

Pelaporan data

Kesalahan data dapat disiapkan melalui template laporan data. Karena belum ada kontak publik terverifikasi di repository, pengiriman eksternal masih menunggu keputusan owner.
