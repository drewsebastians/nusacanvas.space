About NusaCanvas

NusaCanvas is a static app for coloring Indonesia's regencies and cities for presentations, reports, and lightweight planning.

Create a map Why region counts differ

Product principles

Static and private

CSV files and project files are processed in your browser. The app has no accounts, backend, analytics, ads, or cloud project storage.

For visual reference

Exported maps are for visual communication. They are not legal boundary records or tools for precise measurements.

Clear about the data

The map uses a 2020 boundary snapshot with 519 mapped regions. Current administrative information is kept in a separate 2025 region list.

Publisher identity

This repository is prepared to be maintained as drewsebastians/nusacanvas.space. No public email address has been verified in the repository settings. The contact page therefore provides a report template that you can copy or download.

Not available yet

The app does not yet have a custom domain, search indexing, Google AdSense, analytics, user accounts, or built-in datasets for topics such as population, elections, or travel.
