Changelog

A summary of important changes recorded during the staging batches.

Batch 1

  • Added an allowlisted static build and a basic CI workflow.
  • Moved staging to Cloudflare Workers Static Assets with noindex rules.
  • Reduced startup work by loading detailed boundaries only when requested.
  • Added SVG and PNG exports, a PNG fallback, and automated performance limits.
  • Added the version 1 stable region list: 38 provinces, 519 region rows, and 53 records that need more evidence.
  • Added project schema 1.1 with a migration report and prototype-pollution protection.
  • Added a repeatable data process, source inventory, license manifest, data-difference report, and reproducibility check.
  • Added trust pages for About, Contact, Privacy, Terms, Sources and licenses, Data method, Limitations, Changelog, and the region-count guide.

Batch 2 development record

  • The Batch 2 preflight defined the spreadsheet-to-map contract, synthetic import examples, privacy threat model, and resource limits.
  • The Batch 2 import work added local paste, CSV, and TSV previews, column selection, Indonesian and international number formats, and support for older CSV files.
  • XLSX import was added as a locally loaded module with the MIT-licensed reader, sheet selection, ZIP and macro checks, embedded-item and external-link checks, and the same data preview used by other formats.
  • The second matching engine added clear row statuses, local suggestions for uncertain names, manual match and ignore controls, reset controls, and project-specific corrections.
  • Prompt 5 added the Input, Match, Visualize, and Export workflow, a local sample, Basic and Advanced modes, a sortable data table, and two-way links between the table and map.
  • Prompt 6 added category colors, equal ranges, ranked groups, manual ranges, and colors around a midpoint. These use versioned local palettes and the same legend and no-data style in the map and export.
  • Prompt 7 added shared export settings, safe metadata, national and current-view scope, A3, local raster PDF, match-report CSV, and saved color and export settings in project JSON.
  • At that point, later Batch 2 work still included deterministic value classes, a shared legend, the second export format, and expanded project saving.
  • The custom domain, search indexing, analytics, AdSense, user accounts, and backend remained disabled.
