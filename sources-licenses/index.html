Sources and licenses

This page summarizes the sources used by the app. Full records are available in ATTRIBUTION.md, data/README.md, data/sources/source-inventory-v1.json, and data/license-manifest-v1.json.

Map boundaries

Production boundaries use geoBoundaries Indonesia ADM2 from the HDX/OCHA COD-AB Indonesia lineage. ADM2 means second-level administrative boundary data. The selected source represents a 2020 snapshot with boundary ID IDN-ADM2-22746128 and 519 mapped shapes.

The license recorded in the source metadata is CC BY 3.0 IGO / CC BY-IGO. The required attribution must remain in the app and its exports.

Administrative information

The 2025 region list keeps current administrative information separate from the 2020 boundaries. It refers to Kepmendagri 300.2.2-2138/2025 and amendment 300.2.2-2430/2025, but it does not redistribute the official attachments. An official-code field stays empty when there is not enough source evidence for that row.

Libraries and assets

Leaflet 1.9.4 is included locally for map display under the BSD 2-Clause License. The XLSX reader read-excel-file 9.3.1 is included locally, loaded only when an XLSX file is selected, and licensed under the MIT License. The app creates PDFs in the browser without a CDN or an additional PDF library. Development tools such as Playwright, axe-core, and Wrangler are not included in the files sent to app users.

License checks

The latest production source-and-license check recorded 6 source records and 18 licensed assets. A file being publicly available online is not treated as permission for commercial use or redistribution.
