Sumber dan lisensi

Halaman ini merangkum sumber yang digunakan aplikasi. Catatan lengkap tersimpan di `ATTRIBUTION.md`, `data/README.md`, `data/sources/source-inventory-v1.json`, dan `data/license-manifest-v1.json`.

Geometri peta

Geometri produksi memakai geoBoundaries Indonesia ADM2 dari lineage HDX/OCHA COD-AB Indonesia. Snapshot yang dipakai merepresentasikan tahun 2020 dengan boundary ID `IDN-ADM2-22746128` dan 519 fitur geometri.

Lisensi yang tercatat di metadata sumber: CC BY 3.0 IGO / CC BY-IGO. Atribusi wajib dipertahankan pada aplikasi dan ekspor.

Metadata administrasi

Registry v1 memisahkan metadata saat ini dari geometri 2020. Registry merujuk Kepmendagri 300.2.2-2138/2025 dan perubahan 300.2.2-2430/2025, tetapi tidak mendistribusikan lampiran resmi. Field kode resmi yang belum punya bukti baris tetap dikosongkan.

Library dan aset

Leaflet 1.9.4 disertakan lokal untuk rendering peta dan berlisensi BSD 2-Clause. Package pengujian seperti Playwright, axe-core, dan Wrangler tercatat sebagai dependency pengembangan dan tidak ikut dikirim sebagai bundle runtime aplikasi.
