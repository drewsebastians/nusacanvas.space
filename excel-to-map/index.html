Excel to Map Indonesia

Ubah tabel kabupaten/kota menjadi peta presentasi tanpa mengunggah spreadsheet ke server.

1. Masukkan data

Paste dari Excel/Sheets, atau pilih CSV, TSV, dan XLSX lokal.

2. Cek kecocokan

Mapnesia menunjukkan nama ambigu, tidak ditemukan, dan duplikat agar tidak ada tebakan tersembunyi.

3. Visualkan dan ekspor

Pilih kategori, interval sama, kuantil, batas manual, atau divergen; lalu ekspor SVG, PNG, PDF, atau tabel mapping CSV.

Contoh aman untuk dicoba

Mulai dari contoh CSV sintetis, atau ikuti worked example dua kota. Contoh ini bukan data statistik nyata dan dibuat hanya untuk memperlihatkan alur aplikasi.

Untuk siapa?

Untuk pengguna non-GIS yang perlu membuat visual peta bagi presentasi, laporan internal, atau catatan perencanaan. Data tetap lokal di browser; aplikasi tidak menyediakan akun, kolaborasi, atau cloud project storage.
