Terakhir ditinjau: 2026-07-12

Metodologi data

Batch 1 memilih kejelasan dan stabilitas lebih dulu: peta memakai snapshot geometri yang dipin, registry metadata dipisahkan, dan perubahan data harus melewati diff serta review.

1. Snapshot geometri

Geometri produksi berasal dari geoBoundaries Indonesia ADM2 boundary ID `IDN-ADM2-22746128`, merepresentasikan snapshot 2020. File sederhana dipakai untuk startup normal; file detail lokal hanya dimuat saat pengguna memilih ekspor detail.

2. Matching metadata

HDX tabular ADM2 dipakai untuk enrichment ketika nama ADM2 yang dinormalisasi cocok secara unik. Hasil Batch 1 mencatat 466 match unik dan 53 baris ambigu. Baris ambigu tetap bisa diwarnai, tetapi tidak diberi kode/nama/provinsi resmi secara paksa.

3. Registry dan stable ID

`canonical-provinces-v1.csv` menyimpan 38 provinsi saat ini. `canonical-regions-v1.csv` menyimpan 519 baris canonical region berbasis geometri. `crosswalk-region-ids-v1.csv` menjaga kompatibilitas proyek lama.

4. Penyederhanaan peta

Geometri startup disederhanakan dari file detail dengan Douglas-Peucker per ring pada toleransi `0.018` derajat. Ring kecil dipertahankan sejauh praktis; test mencakup beberapa fixture pulau kecil.

5. Reproducibility dan lisensi

Pipeline offline memvalidasi source inventory, license manifest, checksum, diff data, dan reproduksi dua kali. Hasil terakhir: 10 artifact comparable direproduksi identik dengan hash `a9f6c8612e79d6325dd9415d329851313637c4cf998b54d787f6811637cd13e8`.

6. Data spreadsheet dan matching

Paste, CSV, TSV, dan XLSX memasuki kontrak tabular yang sama. Kode wilayah yang terverifikasi diprioritaskan, lalu kombinasi nama dan provinsi. Nama ambigu, tidak cocok, duplikat, diabaikan, kosong, dan invalid tetap dicatat sebagai status berbeda; tidak ada kecocokan yang dipilih secara diam-diam.

7. Visualisasi dan ekspor

Kategori, interval sama, kuantil, batas manual, dan divergen dihitung deterministik dari baris yang cocok dan valid. Nilai kosong bukan nol. Legenda yang sama dipakai peta dan ekspor. SVG, PNG, PDF raster, serta mapping CSV dibuat lokal di browser dengan atribusi boundary wajib.

Review topology

Batch 1 tidak mengklaim validasi topology legal-grade untuk overlap/gap. Validasi mendalam dengan native GIS tetap menjadi review gate sebelum penggantian boundary.
